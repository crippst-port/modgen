ACTIVITY FIELD SPECIFICATIONS:
For EACH activity in the activities array, include these fields:
- type: (required) The activity type from the list above
- name: (required) Descriptive title of the activity
- intro: (optional) Description or context for the activity
- url: (required for 'url' type only) Full URL starting with http:// or https:// (e.g., "https://example.com/article")
- chapters: (required for 'book' type only) Array of chapter objects with 'title' and 'content' fields

EXAMPLE FORMATS:
{"type": "url", "name": "Read Article on Topic X", "intro": "Background material", "url": "https://example.com/article"}
{"type": "quiz", "name": "Knowledge Check", "intro": "Test your understanding"}
{"type": "forum", "name": "Discussion: Topic X", "intro": "Share your thoughts"}
{"type": "book", "name": "Learning Guide", "chapters": [{"title": "Chapter 1", "content": "..."}, {"title": "Chapter 2", "content": "..."}]}
{"type": "label", "name": "Important Note", "intro": "Information for students"}
